# TODO app

Sample application in Node.js

## Requirements
- [Node.js](https://nodejs.org/)
- [SQLite3](https://www.sqlite.org/index.html)

## Installation & Run

```console
$ git clone https://github.com/ser1zw/todoapp-sample.git
$ cd todoapp-sample
$ npm install
$ sqlite3 db/todoapp.db < db/init.sql
$ npm start
```
